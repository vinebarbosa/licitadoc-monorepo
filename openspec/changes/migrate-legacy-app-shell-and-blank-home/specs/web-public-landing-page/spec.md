## ADDED Requirements

### Requirement: Public landing header MUST omit theme switching controls
The public landing page header MUST remain focused on public navigation and call-to-action controls and MUST not expose a theme switch or other theme-changing control.

#### Scenario: Visitor opens the landing page header
- **WHEN** a visitor opens the public landing route
- **THEN** the landing header still exposes the public navigation and primary CTA controls
- **AND** the visitor does not see a theme switch or theme toggle control in that header
